# Carr(In)hos

O **Carr{In}hos** é uma plataforma web de aluguel de veículos icônicos da cultura pop, desenvolvida durante o **Treinamento da IN Junior de 2026.2**.

O projeto (Alugue um Carr{in}ho 26.2) tem como objetivo aplicar os conceitos fundamentais de desenvolvimento front-end, consumo de APIs e trabalho colaborativo utilizando Git e GitHub, por meio da construção de uma aplicação dinâmica.

## Sobre o projeto

A plataforma permite que os usuários explorem veículos famosos de filmes, séries e desenhos animados, consultem informações sobre cada veículo, verifiquem sua disponibilidade e realizem reservas de forma simples.

## Funcionalidades

* Página inicial com veículos em destaque
* Catálogo completo de veículos
* Busca por nome ou obra
* Filtros por categoria e disponibilidade
* Histórico de reservas
* Interface responsiva
* Consumo de API para exibição dinâmica dos veículos
